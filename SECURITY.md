# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.8.x   | :white_check_mark: |

## Reporting a Vulnerability

To report a vulnerability, please
[create an issue](https://github.com/integreat-io/integreat/issues) in this
repo.
