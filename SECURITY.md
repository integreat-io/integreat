# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

To report a vulnerability, please
[create an issue](https://github.com/integreat-io/integreat/issues) in this
repo.
